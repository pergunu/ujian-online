Bantuan & Panduan

Panduan Penggunaan

  1. Pergi ke menu Pencarian
  2. Masukkan kata kunci pencarian (nomor surat, perihal, pengirim/penerima)
  3. Anda dapat memfilter berdasarkan jenis dokumen, kategori, atau rentang tanggal
  4. Klik tombol Cari Dokumen untuk menampilkan hasil
  5. Klik pada dokumen yang ingin dilihat untuk menampilkan detail

  1. Klik tombol WhatsApp di pojok kanan bawah layar
  2. Anda akan diarahkan ke aplikasi WhatsApp
  3. Kirim pesan Anda ke admin
  4. Admin akan merespon pesan Anda secepat mungkin

  1. Temukan dokumen yang ingin dilihat melalui menu pencarian
  2. Klik tombol untuk melihat pratinjau dokumen
  3. Untuk dokumen PDF dan gambar akan langsung ditampilkan
  4. Untuk dokumen Word akan muncul pesan untuk mengunduh terlebih dahulu
  5. Klik tombol untuk mengunduh dokumen
